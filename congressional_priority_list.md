# CONGRESSIONAL READING ROOM PRIORITY LIST
## Ranked by "Predator Name Reveal" Score
### Maximizing the number of predator names revealed per document viewed

**Date:** February 14, 2026
**Purpose:** Prioritize DOJ reading room time to maximize identification of redacted perpetrators
**Scoring:** `REVEAL_SCORE = estimated_redacted_names × crime_severity × novelty_factor`

- **crime_severity:** 3 = rape/CSAM/child trafficking, 2 = adult trafficking/assault, 1 = financial crime/complicity
- **novelty_factor:** 2.0 = name completely unknown, 1.5 = partially identified from context, 1.0 = already identified
- **estimated_redacted_names:** Count of likely person-name redactions per document

**Reading Room Rules:** No electronic devices. Handwritten notes only. Documents organized by EFTA number. Request unredacted versions.

## METHODOLOGY

### Data Sources
- **full_text_corpus.db:** 1,380,941 documents, 2,731,825 pages, FTS5 indexed
- **redaction_analysis_v2.db:** 2,587,102 redactions, 49,153 name-type entities
- **10+ investigation reports** with 400+ key EFTA citations
- **CONGRESS_RAW_EFTA_LIST.md:** 250+ EFTAs categorized by crime type
- **CONGRESSIONAL_READING_GUIDE.md:** Tiered reading list with specific instructions

### Automated Scoring (Layer 1)
- FTS5 search identified 83,567 crime-keyword pages across 24,982 documents
- Redaction density joined from 874,208 page-level redaction records
- Name-proximity patterns (Mr./Ms./Dr. + redaction, "trafficked to," "raped by," etc.)
- Document-level aggregation: crime pages × crime severity × redaction density

### Manual Enrichment (Layer 2)
- 111 EFTAs manually enriched with crime context, redacted name estimates, and novelty factors
- Sources: 10 investigation sessions, 70+ reports, 400+ EFTA citations
- Each enriched EFTA includes: description, crime keywords, estimated redacted names, severity, novelty, source reports

### Final Scoring (Layer 3)
```
REVEAL_SCORE = estimated_redacted_names × crime_severity × novelty_factor
```
- Enrichment values override automated estimates when available
- Automated score provides 10% boost for documents also flagged by keyword/redaction analysis
- Documents with both high crime severity AND high redaction density receive 20% bonus

### Important Caveats
1. **DS9-11 redaction data is unreliable** — "recovered text" from these datasets is garbled OCR of baked-in redaction bars, not actual hidden content
2. **Name entity counts from extracted_entities are 95%+ noise** — used only for density scoring, not name identification
3. **Enrichment is the primary signal** — automated scoring alone produces bulk legal documents at the top due to page-count bias
4. **Victim privacy** — no real victim names, pseudonym-to-name mappings, or identifying details appear in this document
